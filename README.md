# Game Boy Emulator

## Overview
This emulator provides a full implementation of the Game Boy's functionalities.

## Features
- Z80 CPU emulation
- PPU graphics rendering
- Audio playback
- Key input handling

## Installation
1. Clone the repository
2. Run `npm install`
3. Execute `node main.js` to start the emulator.
